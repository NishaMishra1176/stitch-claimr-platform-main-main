## Universal Navigation Rules Applied to All Pages:

### Header Navigation (Consistent Across All Pages):
- Home → My Risk → Evidence → Legal → Recovery
- Each item has dropdown with relevant pages
- "Escalate Now" CTA button always visible

### Footer Navigation (Consistent Across All Pages):
- Stage-based organization (Entry → Diagnosis → Evidence → Legal → Recovery → Closure)
- Platform map with all major tools
- Recovery journey strip showing progression
- Company and legal links

### Clickability Rules:
- Every card is wrapped in `<a href="...">`
- Every stat/number links to detail page
- Every section header is clickable
- Every icon has navigation purpose
- No decorative-only elements

### Instant Closure Paths:
- Every page has direct path to escalation/legal tools
- "Escalate Now" button on every page
- Legal notice generators accessible from any stage
- Nodal officer directory linked from all relevant pages

## Amazon Reconciliation Integration:
- Amazon issues flow through same stages as other issues
- Entry via Financial Triage or Panic Interceptor
- Processed through Forensic → Evidence → Escalation → Recovery
- Specific Amazon terminology mapped to general logistics terms
- Same recovery engine works for Amazon reconciliation issues

## Conversion Optimization Features:
- Panic-first approach (not curiosity-based)
- Forced decision funnels (not browsing)
- Loss visualization (showing money at risk)
- SLA countdown timers (creating urgency)
- Evidence preservation (locking proof before it expires)
- Legal escalation (forcing written response)
- Recovery tracking (monitoring money coming back)
