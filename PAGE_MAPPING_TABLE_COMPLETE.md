## Navigation Flow Rules

### Global Header Navigation (All Pages)
- Home → My Risk → Evidence → Legal → Recovery → Escalate Now
- Each item has dropdown with relevant pages
- Sticky "Escalate Now" CTA on every page

### Footer Navigation (All Pages)
- Stage-based organization (Entry → Diagnosis → Recovery → Closure)
- No repetition of header navigation
- Trust and company information

### Instant Closure Paths (All Pages)
- Every page has direct path to escalation/legal tools
- "Escalate Now" button available on every page
- Users can jump to closure without going through full flow

## Amazon Reconciliation Integration
Amazon reconciliation issues flow through the same stages:
- Entry via Financial Triage or Panic Interceptor
- Classification as "Amazon Settlement Mismatch"
- Processed through Forensic → Evidence → Escalation → Recovery
- Same tools and workflows as other courier issues

## Conversion Optimization Features
- Panic-first approach (not curiosity-based)
- Forced decision funnels (not browsing)
- Loss visualization (showing money at risk)
- SLA countdown timers (creating urgency)
- Evidence preservation (locking proof before it expires)
- Legal escalation (forcing written response)
- Recovery tracking (monitoring money coming back)
